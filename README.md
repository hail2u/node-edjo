edjo
====

Convert CSS file into Every Declaration Just Once style.


INSTALL
-------

    $ npm install -g hail2u/node-edjo


USAGE
-----

Convert:

    $ edjo input.css > output.css

Dump declarations as JSON:

    $ edjo --dump input.css > output.json
